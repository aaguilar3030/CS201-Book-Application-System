Hello this program (Book Application) is used to organize books you have read.
0. The program will use a list of books you have previously read to organize them on the basis of whether they are printed books or audio books.
1-2. When adding a book to the list it will begin with the asking for user input for, the title, name of the author, genre, and pages/length in minutes (depending on if the book is a printed or audiobook).
3. Then The program allows the user to see the last six books that were added to the list. 
4. Next it allows to see the total cost of printed and audio books with the average page count and hour length of the books. It also allows the user to see the total cost of the list.
5. Next all genres found within the list are displayed.
6. The program displays the last three audio books in the list.
7. Displays the full list of books found within the list.
8. Is used to exit the program.
